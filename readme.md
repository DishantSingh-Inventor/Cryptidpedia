# Cryptidpedia Website
### What this project is
This is a simple multi-page website about cryptids. It has pages for some famous monsters like Bigfoot, Nessie, Mothman, Kraken, Chupacabra, Yeti, Jersey Devil and the Mongolian Death Worm. The homepage has cards for each of them with an explore button.
I wanted to make something that looks like a fan site with cool myths and legendary beasts because I really like reading about them.
### Why I made this
I made this because I wanted to practice HTML and CSS. I am still very new to this and wanted to see if I can link different pages together without breaking everything. Also cryptids are super interesting and fun to read about so making a site on them was not boring.
### How I made this
I used basic HTML, CSS and a tiny bit of JavaScript.
* HTML is for the layout and all the information about the beasts
* CSS is for the dark theme and making it look a bit modern
* JavaScript is just for the mobile navigation menu when the screen gets small
I used some AI to help me fix errors in JavaScript and suggest some links for the images when I couldn't find good ones from chrome.
### What I struggled with
The hardest part was definitely CSS layout. Sometimes the boxes would not align properly and some text would go out of the screen. I also struggled with JavaScript because I didn't know how to make the hamburger menu work on mobile and spent a lot of time debugging it. 
Also managing all the different HTML files and making sure the headers are same everywhere was a bit tedious.
### What I learned
From this project I learned:
* How to link multiple HTML pages
* Basic responsiveness for mobile screens
* How to use CSS grid for card layouts
* Finding and linking image URLs from the web
It is not perfect but it is my own work.
